O git é uma ferramenta criada em 2005 e consiste em um sistema de versionamento de código distribuído. Foi criado pelo Linus Torvalds. 

**Importante**: Software não é feito se não for colaborativo, não é feito sozinho. 

Neste módulo aprendi comandos importantes para um bom desempenho do terminal. Exemplo: mudar pastas, listar as pastas, criar pastas/arquivos e deletar pastas;

No windows: cd, dir, mkdir (criar pasta), del/rmdir;

cls (limpar o terminal); ls (listar as chaves); ctrl + l

O SHA1 (Secure Hash Algorithm) é um conjunto de funções criptografadas projetadas pela NASA. A encriptação gera um conjunto de caracteres identificador de 40 dígitos. 

A chave SSH é uma forma de se estabelecer uma conexão segura e encriptada entre duas máquinas. 

git init (mover arquivos); git add; git commit;

git status (status do arquivo)

o git init também inicializa um conceito de git chamado repositório;

**Importante lembrar**: Servidor remoto x ambiente de desenvolvimento / máquina local x nuvem (github).
